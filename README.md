# Forêts françaises, en quête d'avenir — site

Site vitrine du documentaire **« Forêts françaises, en quête d'avenir »** réalisé par Camille Geoffray (54 min, 2022).

Objectif : permettre aux associations, cinémas et réseaux d'environnement de contacter Camille pour organiser des projections-débats.

## Structure

```
.
├── code/        # Site production (HTML/CSS/JS vanilla, prêt Vercel)
├── assets/      # Photos sources (Camille / PNRVN)
└── handoff/     # Bundle Claude Design (référence design system)
```

## Déploiement

Le site est déployé sur Vercel depuis le dossier `code/`. Voir `code/README.md` pour les détails techniques.
